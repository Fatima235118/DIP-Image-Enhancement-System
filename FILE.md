🎯 Project Description

This project enhances low-quality images using MATLAB. It applies image processing techniques such as transformations, sampling, quantization, and histogram processing to improve image quality

⚙️ Features

* RGB to Grayscale conversion
* Image resizing (sampling)
* Bit depth reduction (quantization)
* Geometric transformations (rotation, translation)
* Intensity transformations (gamma, log, negative)
* Histogram equalization

🔁 Pipeline

Input Image → Preprocessing → Transformations → Enhancement → Output Image

How to Run

1. Open MATLAB
2. Place image in project folder
3. Run main script:

matlab
main

4. View enhanced output images

Q/A
Q1:Why histogram equalization improves contrast?
A: It spreads intensity values across full range.

Q2:Effect of gamma?
A: Controls brightness (γ<1 bright, γ>1 dark).

Q3:Quantization effect?
A: Reduces image quality and detail.

Q4:Reversible transformation?
A: Geometric transformations (if parameters known).

Q5:Effect on spatial structure?
A: Changes pixel positions in image.
